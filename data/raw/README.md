# data/raw — Bloomberg Exports

These CSV files are Bloomberg Desktop API exports and are **tracked in version
control** so that teammates can clone the repo and run the pipeline immediately
without manually downloading data.

---

## Files

| File | Tickers | Date range | Frequency |
|---|---|---|---|
| `asml_adr_vols.csv` | ASML US Equity (ADR) | 2016-01-04 → 2025-12-31 | Daily |
| `asml_loc_vols.csv` | ASML NA Equity (local, Amsterdam) | 2016-01-04 → 2025-12-31 | Daily |
| `eurusd_fx_vols.csv` | EURUSD Curncy | 2016-01-01 → 2025-12-31 | Daily |

---

## Column schemas

### `asml_adr_vols.csv` and `asml_loc_vols.csv`

| Column | Bloomberg field | Description |
|---|---|---|
| `Date` | — | Trade date |
| `ATM_1M` | `CALL_IMP_VOL_30D` | 1-month ATM implied vol (%) |
| `P25_1M` | `1M_PUT_IMP_VOL_25DELTA_DFLT` | 1-month 25-delta put implied vol (%) |
| `C25_1M` | `1M_CALL_IMP_VOL_25DELTA_DFLT` | 1-month 25-delta call implied vol (%) |
| `ATM_3M` | `3MO_CALL_IMP_VOL` | 3-month ATM implied vol (%) |
| `ATM_1Y` | `12MO_CALL_IMP_VOL` | 1-year ATM implied vol (%) |
| `ADR_SPOT` / `LOC_SPOT` | `PX_LAST` | Closing spot price |

### `eurusd_fx_vols.csv`

| Column | Bloomberg field | Description |
|---|---|---|
| `Date` | — | Trade date |
| `ATM_1M` | `EURUSD1M BGN Curncy PX_LAST` | 1-month ATM vol (%) |
| `RR_1M` | 25-delta risk reversal | σ_25c − σ_25p (%) |
| `BF_1M` | 25-delta butterfly | (σ_25c + σ_25p)/2 − ATM (%) |
| `ATM_3M`, `RR_3M`, `BF_3M` | same, 3-month tenor | — |
| `ATM_1Y`, `RR_1Y`, `BF_1Y` | same, 1-year tenor | — |
| `FX_SPOT` | `PX_LAST` | EUR/USD closing spot rate |

> **Units**: all vol columns are in **percentage points** as delivered by
> Bloomberg (e.g. `10.5` = 10.5%). The loader converts them to decimal form
> (`0.105`) internally — **do not pre-convert**.

## Re-pulling from Bloomberg

If you need to refresh or extend the data, use the existing BDH helper:

```python
# From the project root (Bloomberg Terminal must be open and logged in):
python scripts/bdh_generic.py
```

or export directly from the Bloomberg Excel Add-In using the formulas
embedded in the original spreadsheet (visible as comments in the trailing
columns of each CSV).

The following are git-ignored and must **not** be committed:
